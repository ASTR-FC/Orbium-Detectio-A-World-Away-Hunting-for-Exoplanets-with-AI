L'Enjeu

Les données de plusieurs missions spatiales d'arpentage d'exoplanètes, telles que Kepler, K2 et TESS, ont permis la découverte de milliers de nouvelles planètes en dehors de notre système solaire. Cependant, la plupart de ces exoplanètes ont été identifiées manuellement. Avec les progrès de l'intelligence artificielle et de l'apprentissage automatique (IA/ML), il est désormais possible d'analyser automatiquement de grands ensembles de données collectées par ces missions pour identifier les exoplanètes de manière plus efficace.

L'identification des exoplanètes est un domaine en pleine expansion dans l'exploration astronomique. Les missions Kepler, K2 et TESS ont fourni des données précieuses, mais leur analyse reste en grande partie manuelle. L'automatisation de cette analyse grâce à l'IA/ML peut accélérer les découvertes et rendre ce domaine plus accessible.

Notre Mission : Répondre à la Problématique

Notre mission est de créer un modèle d'IA/ML formé sur les ensembles de données d'exoplanètes open-source de la NASA, capable d'analyser de nouvelles données pour identifier avec précision les exoplanètes. Nous avons développé une solution qui utilise des algorithmes d'apprentissage automatique pour analyser les données de transit et identifier les exoplanètes de manière automatisée.

Nous avons utilisé des ensembles de données des missions Kepler, K2 et TESS pour entraîner notre modèle. Notre solution permet non seulement d'identifier les exoplanètes, mais aussi de faciliter l'interaction des utilisateurs avec les données grâce à une interface web intuitive.

Notre modèle prend en compte plusieurs variables, telles que la période orbitale, la durée de transit, et le rayon planétaire, pour classer chaque point de données comme une exoplanète confirmée, un candidat planétaire ou un faux positif. Nous avons également intégré des fonctionnalités pour permettre aux utilisateurs de télécharger de nouvelles données et de visualiser les résultats de manière claire et concise.

Manpage de l'Application Web

Description

Notre application web permet aux utilisateurs d'analyser des ensembles de données de transit pour identifier les exoplanètes. Elle offre une interface intuitive pour télécharger des fichiers de données, visualiser les résultats et interagir avec les prédictions du modèle.

Fonctionnalités

  • Upload de données : Téléchargez vos fichiers CSV contenant les données de transit.
  • Analyse en temps réel : Obtenez des prédictions instantanées via notre modèle entraîné.
  • Visualisation des résultats : Visualisez les résultats avec des graphiques et des statistiques.
  • Mise à jour du modèle : Ré-entraînez le modèle avec de nouvelles données.
  • Tweaking des hyperparamètres : Ajustez les paramètres du modèle pour les utilisateurs avancés.
  • Export des résultats : Téléchargez les prédictions au format CSV ou PDF.

Utilisation

  1. Accédez à la page d'analyse via le menu de navigation.
  2. Téléchargez un fichier CSV contenant vos données de transit.
  3. Visualisez les résultats de l'analyse.
  4. Exportez les résultats si nécessaire.

Public Cible

Notre application est conçue pour les chercheurs qui veulent classer de nouvelles données, ainsi que pour les novices dans le domaine qui veulent interagir avec les données des exoplanètes et ne savent pas par où commencer.
